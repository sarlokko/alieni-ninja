# Alieni Ninja

Landing page sci-fi che unisce **alieni extraterrestri** e **guerrieri ninja** in un'élite intergalattica.

## Struttura

```
public/
  index.html    # Landing page
  css/style.css # Stili sci-fi
  js/main.js    # Bottone "Lancia missione"
package.json    # Script npm start
```

## Avvio locale

```bash
npm install
npm start
```

Il sito sarà disponibile su [http://localhost:3000](http://localhost:3000).

## Funzionalità

- Design dark con effetto starfield e gradienti neon
- Sezioni Missione e Squadra
- Bottone **Lancia missione** con messaggi random a tema alieni/ninja
